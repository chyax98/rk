🗺️ 图表引擎展示

RenderKit 支持四种图表引擎：Mermaid（客户端 CDN）、D2（客户端 WASM）、Graphviz（客户端 CDN）、PlantUML（服务端 Kroki SSR）。

Mermaid 流程图

flowchart TD A([开始]) --> B[填写注册表单] B --> C{验证通过?} C -->|是| D[创建账号] C -->|否| E[显示错误] E --> B D --> F[发送验证邮件] F --> G([完成])

D2 架构图

Agent: { shape: rectangle label: "AI Agent" } CLI: { shape: rectangle label: "renderkit CLI" } Server: { shape: rectangle label: "Next.js Server" } Browser: { shape: rectangle label: "Chrome" } DB: { shape: cylinder label: "SQLite" } Agent -> CLI: "rk push doc.html" CLI -> Server: "POST /api/artifacts" Server -> DB: "store artifact" Server -> Browser: "render HTML + WC" Browser -> Agent: "rk feedback → comments JSON"

digraph RenderKit { rankdir=LR; node [shape=box, style=rounded, fontname="sans-serif"]; web [label="@renderkit/web", fillcolor="#dbeafe", style="rounded,filled"]; components [label="@renderkit/components", fillcolor="#dcfce7", style="rounded,filled"]; design [label="@renderkit/design", fillcolor="#fef9c3", style="rounded,filled"]; cli [label="@renderkit/cli", fillcolor="#fce7f3", style="rounded,filled"]; web -> components; web -> design; cli -> web [label="push/feedback", style=dashed]; components -> design [label="consumes tokens"]; }

PlantUML 时序图 （需服务端 Kroki SSR）

@startuml skinparam sequenceArrowThickness 2 skinparam roundcorner 10 actor Agent participant "RenderKit CLI" as CLI participant "Server" as SRV participant "Browser" as WEB actor Human Agent -> CLI : rk push report.html CLI -> SRV : POST /api/artifacts SRV --> CLI : { artifactId, url } CLI --> Agent : artifact URL Human -> WEB : 打开 URL 审阅 Human -> WEB : 添加评论 WEB -> SRV : POST /api/comments Agent -> CLI : rk feedback report.html CLI -> SRV : GET /api/comments SRV --> CLI : [{ text, anchor }] CLI --> Agent : comments JSON Agent -> Agent : 分析评论，修改 HTML Agent -> CLI : rk push report.html (更新) @enduml
